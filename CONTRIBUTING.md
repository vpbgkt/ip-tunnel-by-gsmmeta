# Contributing to IP Tunnel by GsmMeta

Thank you for your interest in contributing to IP Tunnel by GsmMeta!

## How to Contribute

### Reporting Bugs

1. Check if the bug has already been reported in [Issues](https://github.com/yourusername/ip-tunnel-gsmmeta/issues)
2. If not, create a new issue with:
   - Clear title and description
   - Steps to reproduce
   - Expected vs actual behavior
   - System information (Windows version, Python version)
   - Screenshots if applicable

### Suggesting Features

1. Open an issue with the `enhancement` label
2. Describe the feature and its use case
3. Explain why it would be useful

### Pull Requests

1. Fork the repository
2. Create a new branch: `git checkout -b feature/my-feature`
3. Make your changes
4. Test thoroughly
5. Commit with clear messages
6. Push to your fork
7. Submit a pull request

### Code Style

- Follow PEP 8 guidelines
- Use meaningful variable names
- Add docstrings to functions
- Comment complex logic
- Keep functions focused and small

### Testing

Before submitting:
```bash
python test_system.py  # Run system tests
python main.py         # Manual testing
```

## Development Setup

```bash
git clone https://github.com/yourusername/ip-tunnel-gsmmeta.git
cd ip-tunnel-gsmmeta
pip install -r requirements.txt
python main.py
```

Feel free to open an issue for any questions!
